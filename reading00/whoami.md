TLDR - whoami
==========

Overview
--------

[whoami] is a command that prints the user name associated with the current effective user ID

Examples
--------

- **Display** user *id*:

        $ whoami

- **Display** this *help* and exit:

        $ whoami --help

- **Display** *version* information and exit:

        $ whoami --version
