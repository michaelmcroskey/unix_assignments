TLDR - groups
==========

Overview
--------

[groups] is a command that prints the groups a user is in

Examples
--------

- **Display** the *groups* a user is in:

        $ groups

- **Display** this *help* and exit:

        $ groups --help

- **Display** *version* information and exit:

        $ groups --version
