TLDR - ps
==========

Overview
--------

[ps] is a command that report a snapshot of the current processes

Examples
--------

- **Report** a snapshot of the current processes:

        $ ps

- **Print** a process tree:

        $ ps -ejH
        $ ps axjf

- **Select** all processes:

        $ ps -A

- **Restrict** selection to running processes:

        $ ps -r
