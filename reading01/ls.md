TLDR - ls
==========

Overview
--------

[ls] is a command that lists directory contents

Examples
--------

- **Display** list directory:

        $ ls

- **Display** list as *human readable*:

        $ ls -l
		$ ls -s

- **Print** the *index number* of each file:

        $ ls -i

- **Print** the allocated *size* of each file:

        $ ls -s
