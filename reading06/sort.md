TLDR - sort
==========

Overview
--------

[sort] is a command that is used to arrange files alphabetically and numerically

Examples
--------

- *Sort* contents alphabetically (default)

	$ ls | sort
	
- *Sort* contents numerically (-nr for reverse)
		
	$ sort -n file

- *Remove* duplicates
		
	$ sort -u file
