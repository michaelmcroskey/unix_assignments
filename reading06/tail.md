TLDR - tail
==========

Overview
--------

[tail] is similar to head but prints the last 10 lines of a file 

Examples
--------

- *Display* last 35 lines of cat file
		
	$ tail -n 50 cat.txt

- *Output* and append as file grows

	$ tail -f

- *Display* last 16 bytes of file

	$ tail -c16 dog.txt
